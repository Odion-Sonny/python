28. What are modules in Python?

A module is a file that includes a set of various functions and statements that can be added to an application.

They are basically of two types:

  • Built-in modules: Pre-installed with Python (e.g., math, os, sys)
  • User-defined modules: Created by developers for specific needs
# Importing a module import math from datetime import datetime # Using module functions result = math.sqrt(16) current_time = datetime.now()
29. What is a Python PATH?

Python PATH is an environment variable used to import modules and check for the presence of variables present in different directories. It tells Python where to look for modules when importing.

32. What is lambda in Python?

Lambda is a small anonymous function used in Python as an inline function.

Syntax: lambda arguments : expression

# Basic lambda function square = lambda x: x ** 2 print(square(5)) # Output: 25 # Lambda with multiple arguments add = lambda x, y: x + y print(add(3, 4)) # Output: 7 # Using lambda with map() numbers = [1, 2, 3, 4, 5] squared = list(map(lambda x: x**2, numbers)) print(squared) # Output: [1, 4, 9, 16, 25]
33. How is memory managed in Python?

In Python, memory is managed using the Python Memory Manager. The manager allocates memory in the form of a private heap space dedicated to Python. All objects are stored in this heap, and due to its private nature, it is restricted from direct programmer access.

Key aspects:

  • Automatic garbage collection
  • Reference counting
  • Memory pools for efficient allocation
  • Private heap space for Python objects
